tac0de ai

A “mastermind” architecture that treats control as a design problem. It separates planning from execution, and keeps Deep mode optional.

  • Default: fast mode (no deep trace, no extra latency)
  • Optional: Deep mode (structured trace only, no side effects)
  • Goal: support human thinking, not replace it

tac0de-cascade

A Deep Thinking protocol language for AI. Not a programming language for execution, but a format that forces depth: contradiction checks, counterfactuals, dignity/harm checks, and synthesis.

Deep mode contract

  • Deep mode is OFF by default.
  • When ON: generate a Trace (structured JSON). No external actions.
  • When OFF: do nothing extra (zero impact).
